# Curriculum Files

Each `day-XXX.json` file is one daily assignment loaded by the Spring Boot backend.

Sections inside every file:
- `notes`
- `conceptualQuestions` for MCQs
- `writtenConceptQuestions`
- `businessScenarios`
- `codingTasks`
- `sqlPractice`
- `springBootScenarios`
- `angularQuestions`
- `dsaPractice` with statement, constraints, examples, hints, and expected complexity

Edit these files directly to customize the course content.
